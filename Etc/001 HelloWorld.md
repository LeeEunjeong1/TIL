# HelloWorld 어원
프로그래밍 예제파일에 항상 첫번째로 있는 "Hello World!"
왜.. 하필  Hello World 일까...!!<br>
>1978년에 브라이언 커니핸과 데니스 리치가 쓴 "The C Programming Language" 교재의 첫 번째 예제가 화면에 "Hello, world!"를 출력하는 것이었고, 이것이 유명해지면서 모든 프로그래밍 언어의 첫번째 예제로 굳어지게 되었다.<br>
